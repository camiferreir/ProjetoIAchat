README – Royal Pet Care 🐾
1. Introdução

O projeto Royal Pet Care consiste em uma Landing Page moderna desenvolvida para um Pet Shop especializado em Banho e Tosa de Alto Padrão.

O objetivo principal é transmitir profissionalismo, exclusividade, confiança e cuidado, oferecendo uma experiência agradável ao usuário e incentivando o agendamento dos serviços por meio de uma interface moderna e intuitiva.

A aplicação foi desenvolvida utilizando apenas HTML5, CSS3 e JavaScript Vanilla, sem a utilização de frameworks ou bibliotecas externas, conforme solicitado na atividade.

2. Objetivos do Projeto
Desenvolver uma Landing Page responsiva para um Pet Shop Premium.
Aplicar conceitos de UX (Experiência do Usuário).
Utilizar HTML5 semântico.
Criar uma interface moderna em Dark Mode.
Aplicar animações e efeitos visuais com CSS.
Desenvolver validações utilizando JavaScript.
Criar um formulário funcional de agendamento.
3. Tecnologias Utilizadas
HTML5

Responsável pela estruturação do conteúdo da página.

CSS3

Responsável pela estilização, layout responsivo e animações.

JavaScript Vanilla

Responsável pelas validações e interações do usuário.

9. Funcionalidades JavaScript
Validação do Formulário

O sistema verifica se:

Nome do tutor foi preenchido.
Nome do pet foi preenchido.
Telefone foi preenchido.
Data foi selecionada.
Serviço foi escolhido.

Caso algum campo esteja vazio:

Preencha todos os campos.
Máscara de Telefone

Durante a digitação:

(51) 99999-9999

A formatação é aplicada automaticamente.

Controle de Datas

O usuário não pode selecionar uma data anterior ao dia atual.

Objetivo:

Evitar agendamentos inválidos.

Mensagem de Sucesso

Após o envio:

Agendamento realizado com sucesso para o pet Thor!

A mensagem aparece sem recarregar a página.
