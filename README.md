# 🐾 Royal Pet Care

## Sobre o Projeto

Royal Pet Care é uma Landing Page desenvolvida para um Pet Shop especializado em Banho e Tosa Premium. O site foi criado com foco em experiência do usuário (UX), design moderno e facilidade de agendamento dos serviços.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout responsivo
- Design Dark Mode
- Cards de serviços com preços e tempo estimado
- Selo "Mais Vendido"
- Botão de agendamento com efeito neon
- Formulário de agendamento
- Validação de campos
- Máscara para telefone
- Bloqueio de datas anteriores ao dia atual
- Mensagem de confirmação sem recarregar a página

## Estrutura do Projeto

RoyalPetCare/

├── index.html

├── style.css

├── script.js

└── README.md

## Como Executar

1. Baixe os arquivos do projeto.
2. Mantenha todos os arquivos na mesma pasta.
3. Abra o arquivo `index.html` em seu navegador.

## Objetivo

Aplicar os conhecimentos de HTML, CSS e JavaScript na criação de uma Landing Page moderna, responsiva e funcional para um Pet Shop de Alto Padrão.
