# 🐾 Royal Pet Care - Landing Page

## 📌 Sobre o Projeto

A Royal Pet Care é uma Landing Page desenvolvida para um Pet Shop de Alto Padrão especializado em Banho e Tosa.

O projeto foi criado utilizando apenas HTML5, CSS3 e JavaScript Vanilla, seguindo os requisitos da atividade proposta em sala de aula.

A página possui um design moderno em Dark Mode, foco em experiência do usuário (UX), responsividade e elementos visuais que incentivam o agendamento dos serviços.

---

## 🎯 Objetivo

Desenvolver uma Landing Page profissional para apresentar serviços de banho e tosa premium, permitindo que os clientes realizem agendamentos de forma simples e intuitiva.

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
